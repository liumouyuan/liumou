## 关于

还在为出门要不要带伞纠结吗？这个工具可以让你在终端里秒查天气！支持全球4万+城市数据，附带摸鱼模式（显示随机趣味天气提示）。

## 功能

- 🌍 查询任意城市当前天气
- 🌡️ 显示温度/湿度/风速
- 📅 未来3小时简短预报
- 🐟 摸鱼模式（随机显示趣味天气冷知识）

## 安装

```bash
# 1. 克隆仓库
git clone https://github.com/yourname/weather-fetch.git

# 2. 安装依赖
cd weather-fetch
npm install

# 3. 全局链接（可选）
npm link
